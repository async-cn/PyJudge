# C 定身魔法

不知道数了多少只绵羊，Maaike 终于入睡了。第二天早上，Maaike 准备继续上路。然而，他发现他的地图不见了。

在转角处，他发现了一只老鼠，嘴里叼着的正是他的地图。然而，老鼠跑得实在是太快了，Maaike 决定使用 “定身魔法” 将老鼠定住。

Maaike 翻开魔法书，发现有关定身魔法的记载如下：

> 想要使用定身魔法，就一直重复喊出 $123456$ 吧。每个猫咪使用定身咒时需要喊出的数字数量是不同的，这取决于它们的魔力值。只需要喊出与自身魔力值数量相同的数字就可以了。
> 
> 比如魔力值为 $10$ 的猫咪需要喊出的咒语如下：
> 
> $$1234561234$$
> 
> 而魔力值为 $14$ 的猫咪需要喊出的咒语则更长一些，不过依然是重复的：
> 
> $$12345612345612$$
> 
> 魔力值越高，魔法的生效持续时间越长，而需要喊出的咒语也更长。这有时并没有什么好处，比如魔力值为 $10000$ 的猫咪喊完咒语后，想要定身的目标早就跑没影了。
> 
> 不过，千万不要喊多哪怕一个数字，不然的话，你就会把自己定住，那样就麻烦了。

请你编写一个 Python 程序，给定 Maaike 的魔力值，帮助 Maaike 喊出正确的咒语（一行一个数字）。

## 输入格式

一行，一个整数，代表 Maaike 的魔力值 $n$

## 输出格式

$n$ 行，每行一个一位数字，为 $123456123456\cdots$ 中的循环的一位数字。

## 输入输出样例

### 样例输入 #1

```text
10
```

### 样例输出 #1

```text
1
2
3
4
5
6
1
2
3
4
```

## 数据范围

对于所有测试数据，$1\le n\le {10}^5$

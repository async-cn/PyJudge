# B 数绵羊

不知过了多久，Maaike 终于打开了大门。夹杂着雨后的泥土味和青草香味的风扑面而来，这自由的味道是 Maaike 在 Cats 小镇从未感受过的。

经过一段时间的跋涉，Maaike 决定在一个客栈住一晚上。然而，夜晚的蝉鸣不绝于耳，吵得 Maaike 无法入睡。聪明的 Maaike 想到通过数绵羊来使自己入睡。

请你编写一个 Python 程序，帮助 Maaike 数绵羊。给定绵羊数量 $n$，输出第 $1\sim n$ 只绵羊的序号（从 $1$ 开始）。

具体地，输入正整数 $n$，依次输出 $1\sim n$ 之间的所有整数。

## 输入格式

一行，一个正整数 $n$

## 输出格式

$n$ 行，每行一个正整数，代表绵羊的序号。其中第 $i$ 行代表第 $i$ 只羊，序号为 $i$。

## 输入输出样例

## 样例输入 #1

```text
6
```

## 样例输出 #1

```text
1
2
3
4
5
6
```

## 数据范围

对于所有测试数据，$1\le n\le {10}^5$
